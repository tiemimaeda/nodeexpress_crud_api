# Simple CRUD REST API

### Technologies used:
* Node.js
* Express
* UUID 

### Practiced:
* How to build a simple API
* CRUD request methods
* How to use UUID to generate randoms id's

### Routes created:
        .get('/', getUsers);

### Check it out:
